## 주요 기능
**테트리스 게임**
- 모양은 랜덤으로 하나가 나오게 됩니다.
- 숫자를 지정하여 모양을 내릴 위치를 선택할 수 있습니다.
- 한 줄이 채워진는 경우 윗 줄이 아래로 내려고 채워진 줄은 깨져 점수가 추가 됩니다.
- 일정 점수 도달 시 level이 증가하며 level에 따라 기준 점수가 높아집니다.
